Basic:

```js
<FlyTooltip content={<p><strong>Yo! This is a test.</strong><br />I'm a tooltip.</p>}>
    <strong>Hover over me</strong><br />
</FlyTooltip>
```

With exclamation icon:

```js
<FlyTooltip position="bottom" exclamation={true} content={<p><strong>Yo! This is a test.</strong><br />I'm a tooltip.</p>}>
    <strong>Hover over me</strong><br />
</FlyTooltip>
```
